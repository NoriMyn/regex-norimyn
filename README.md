# 🐈‍⬛ Регексы от NoriMyn

> Удобный набор регексов для **SillyTavern**, который помогает очищать контекст от лишних служебных элементов и держать генерацию в порядке.

---

## ✨ Возможности

Это расширение собирает полезные регексы для удаления лишних блоков из общего контекста в **SillyTavern**.

### В набор входят:

- 🧠 **Очистка thinking:**
  Удаляет думалку в формате `<thinking>`.

- 💭 **Очистка think:**
  Удаляет думалку в формате `<think>`.

- 📦 **Инфоблок:**
  Удаляет инфоблок из общего контекста.

- 🎛️ **Панель кнопок:**
  Удаляет панель кнопок из общего контекста, если вы ими пользуетесь.

- 🧹 **Удаление HTML**
  Удаляет HTML, что полезно при генерации артов внутри таверны.

---

## 📌 Зачем это нужно

Служебные блоки, кнопки, HTML и дополнительные элементы интерфейса могут попадать в контекст и мешать качественной генерации.

**🐈‍⬛ Регексы от NoriMyn** помогают:

- убрать лишний мусор из контекста;
- сделать переписку чище;
- снизить влияние служебных вставок на ответы модели;
- упростить работу с интерфейсными элементами и оформлением.

## 🚀 Установка

Установите расширение через встроенный менеджер расширений в **SillyTavern**:

1. Откройте **SillyTavern**
2. Перейдите в **Расширения**
3. Нажмите **Установить расширение**
4. Вставьте ссылку:

   ```text
   https://github.com/NoriMyn/regex-norimyn
   ```

5. Подтвердите установку
6. Перезагрузите **SillyTavern**

После установки:

1. Откройте настройки расширения

4. Используйте только те фильтры, которые нужны именно вам
